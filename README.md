> Record
>
> Record 是 Java 14 引入的一个新特性（在 Java 16 中成为正式特性），它是一种更简洁的方式来创建不可变的数据载体类。
> Record 简介
> 1. 什么是 Record？
> Record 是一种特殊的类，主要用于存储不可变的数据。它自动为你生成：
> + 构造器
> + getter 方法
> + equals() 方法
> + hashCode() 方法
> + toString() 方法
